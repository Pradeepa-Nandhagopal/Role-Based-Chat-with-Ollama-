# Role-Based Chat with Ollama 🤖

## 📌 Project Overview

This project is a role-based conversational AI system built using Python and Ollama.
It allows users to interact with an LLM using different roles such as Python Tutor, Fitness Coach, and Travel Guide.

---

## 🚀 Features

* 🎭 Multiple predefined roles
* 🔁 Role switching during chat
* 🧠 Maintains conversation history (context-aware)
* ➕ Add custom roles dynamically
* ⚡ Streaming responses (real-time output)
* ⏱ Response time tracking
* 🔢 Token count (approx)

---

## 🛠 Tech Stack

* Python
* Ollama (LLM runtime)
* LLaMA3 model

---

## 📂 File Structure

chat.py  → Main application file

## ⚙️ Installation & Setup

### 1. Install Ollama

Download from: https://ollama.com

### 2. Pull Model

```bash
ollama run llama3
```

### 3. Install Python Package

```bash
pip install ollama
```

---

## ▶️ How to Run

```bash
python chat.py
```

---

## 🎮 Commands

* `switch` → Change role
* `quit` → Exit program
* `add` → Add custom role
